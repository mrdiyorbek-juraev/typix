---
title: useEditor
description: Access the Lexical editor instance and empty state
---

`useEditor` provides the raw Lexical editor instance along with a reactive `isEmpty` flag.

## Import

```tsx
import { useEditor } from '@typix-editor/react';
```

## Usage

```tsx
function SubmitButton() {
  const { editor, isEmpty } = useEditor();

  return (
    <button disabled={isEmpty}>
      Submit
    </button>
  );
}
```

### Accessing the Lexical API

```tsx
function MyPlugin() {
  const { editor } = useEditor();

  const handleClick = () => {
    editor.update(() => {
      const selection = $getSelection();
      // Direct Lexical operations
    });
  };

  return <button onClick={handleClick}>Do something</button>;
}
```

## Return value

| Property | Type | Description |
| --- | --- | --- |
| `editor` | `LexicalEditor` | The Lexical editor instance |
| `isEmpty` | `boolean` | Whether the editor content is empty (reactive) |

## isEmpty behavior

The `isEmpty` flag updates reactively whenever the editor content changes. It uses Lexical's `$isRootTextContentEmpty` to determine if the editor has any text content.

## Requirements

Must be used inside a `LexicalComposer` (which `EditorRoot` sets up automatically).
