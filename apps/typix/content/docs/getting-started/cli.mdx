The Typix CLI (`@typix-editor/cli`) lets you install extensions directly from your terminal — no copy-pasting package names.

## Installation

Run without installing using `npx`:

```bash
npx @typix-editor/cli add
```

Or install globally:

```bash
npm install -g @typix-editor/cli
```

## Commands

### `typix add`

Installs one or more extensions. The CLI auto-detects your package manager (`pnpm`, `npm`, `yarn`, or `bun`) from your lockfile and runs the appropriate install command.

**Interactive mode** — run without arguments for a multi-select picker:

```bash
typix add
```

**Direct install** — pass one or more extension names:

```bash
typix add auto-link floating-link
```

**Install everything** — add all available extensions at once:

```bash
typix add --all
```

### `typix list`

Print all available extensions with their package names and descriptions:

```bash
typix list
```
